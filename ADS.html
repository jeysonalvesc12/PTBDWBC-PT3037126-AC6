Análise e Desenvolvimento de Sistemas

O que é?

O curso de Análise e Desenvolvimento de Sistemas é um programa de educação técnica que forma profissionais capacitados para criar, manter e melhorar soluções de software.

Áreas de Atuação

  • Desenvolvimento de aplicações web e mobile
  • Análise e design de sistemas de software
  • Banco de dados
  • Suporte técnico e manutenção de sistemas
  • Engenharia de software

Principais Disciplinas

  • Linguagens de programação
  • Desenvolvimento web (HTML, CSS e JavaScript)
  • Banco de dados e SQL
  • Análise de requisitos
  • Segurança da informação
  • Padrões de design e arquitetura

Competências Desenvolvidas

  • Raciocínio lógico e resolução de problemas
  • Domínio de linguagens de programação
  • Conhecimento de ferramentas e tecnologias atuais
  • Trabalho em equipe e comunicação
